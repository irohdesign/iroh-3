Overview

Round Pond Coffee has a devoted following in Coastal Maine. Their Facebook and Yelp pages are overflowing with rave reviews, villagers chat while waiting for a cappuccino (unthinkable in Round Pond less than 5 years ago) and local artists display their work on the walls.

With the store growing more and more each year, the owners were looking to expand their reach and hired me. I built them a website that would bring in more tourists from neighboring hotspots, show off their menu, and increase their visibility online.

The Challenge

The primary challenge in building this website was the region’s lack of cell reception. To say that Round Pond has poor digital infrastructure would be an understatement. In 2014, Maine’s internet speed ranked at 49th in the country. Most of Maine’s counties top download speeds range between 40-60 percent below the national average.

Complete pages in Maine take minutes to fully load. I designed the site to be functional as soon as possible, even if the page cannot finish loading.

This website needed to be fast, lightweight, and most importantly; fail gracefully.

Mobile Sketches

High Contrast Logo

Branding is front and center. Better legibility if background load is delayed.

Collapsed Menu

Smooth scroll animation for better interactions. Simple, clear navigation across the site with no exterior links to slow down website.

Solid Background

Faster, more consistent load on mobile devices.

Clear Heading

The menu is very easy to identify. Eliminating any extra media makes load faster and helps the user see what's important; the food.

Tab-Based Menu

I built the menu using Bootstrap and jQuery to save valuable screen space. Instead of having to scroll through a long menu, users can focus on what they want and easily browse.

Casual Appearance

I chose a handwritten font for the headings to give a more relaxed feel to the menu. The listings are all still clear and easy to read, but also match the attitude of the shop.

Clear Address

The address is clear and can be copied and pasted quickly, if the user wants to use an alternative navigation app.

Social Media Links

Round Pond Coffee have five star ratings on Facebook and Yelp, so I decided to direct links for users to see the positive reviews quickly.

Interactive Map

I embedded their location with GoogleMaps so users can immediately open directions to the store and travel there, accomplishing one of the three main objectives of this project.

Design Process

Mobile-First Approach

Tourists primarily search for food using their phones. I prioritized the mobile user experience so that customers would be able to find the store while visiting nearby attractions.

Prioritizing User Experience

I used Bootstrap and jQuery to build an interactive menu based on tabs. Users now have access to an entire menu without having to open a separate document or webpage.

The menu's tabs save valuable screen real estate and minimize unnecessary scrolling. The smooth scroll improves user experience and reinforces the availability to the rest of the page.

Branding Choices

Round Pond Coffee is a small, locally-owned business. Customers choose to support their Maine-made coffee blends because they want an authentic experience.

I decided to balance the graphic design, making it modern and easy to use, but not too flashy or trendy. I paid special attention to match the website to the shop’s personal brand and message.

Development

Balancing Speed & Performance

I trimmed down the code and files to keep the website as fast as possible. I used Bootstrap’s framework because the built-in styles and scripts kept the code from getting too dense.

I placed the Javascript and jQuery tags at the bottom of the HTML so the structure and appearance are prioritized. Users can get what they need to find the store, even if the page fails.

I used Google’s development tool, PageSpeed, to better understand where the weight of my website was focused. I deleted all landing page redirects, then minified the CSS, HTML and Javascript files. I optimized the images using Adobe Photoshop’s web exporting tools. I prioritized visible content and emphasized legibility to ensure a fast, functional experience for my users.

Roadblock

I chose to embed Instagram posts geotagged to the store to show off their existing digital presence and give the website a personal touch.

While testing across all the major browsers, I found an issue. Despite the images appearing on all other major browsers, the photos would not display on Safari.

Safari Before the Fix

I looked through Stack Overflow and found other developers with the same issue. Through a combination of the advice on the forum and my own digging, I discovered the issue.

The images failed to load because Instagram’s generated Javascript tags do not include a hypertext transfer protocol (HTTP). This causes links to be unstable across different browsers. By adding the protocol, I fixed the issue for all browsers.

Results

The owners are very happy with the completed project. I launched the site in early 2018, just in time for the Spring Opening.
